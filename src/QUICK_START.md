# Quick Start Guide - Smart Robotic Shopping Cart

## 🎯 Quick Navigation Map

```
HOME SCREEN (Entry Point)
├── Scan Item → SCAN PRODUCT SCREEN
│   ├── Camera Scan (try barcodes: 123456789, 987654321)
│   ├── Manual Entry
│   └── View Product Details → PRODUCT DETAILS
│
├── Search → SEARCH SCREEN
│   ├── Products Tab (search, browse categories)
│   ├── Promotions Tab (view deals)
│   └── Product Results → PRODUCT DETAILS
│
├── Checkout → CHECKOUT SCREEN
│   ├── View Cart Items
│   ├── Select Payment Method
│   └── Complete Purchase
│
├── Start Shopping → SHOPPING LIST SCREEN
│   ├── Scan Product
│   ├── Search Products
│   └── Upload List Options
│
└── Navigation Bar (Bottom)
    ├── Home
    ├── List
    ├── Navigate (Store Map)
    ├── Cart (with item count badge)
    ├── AI Chatbot
    └── Profile → Feature Guide
```

## 🚀 Key Features Quick Access

### Product Scanning
- **From Home**: Tap "Scan Item" (purple button)
- **Test Barcodes**: 
  - `123456789` = Organic Bananas
  - `987654321` = Whole Milk
- **Features**: Camera scan or manual entry

### Smart Search
- **From Home**: Tap "Search" (blue button)
- **Try Searching**: "Organic", "Milk", "Bread"
- **Browse**: Categories (Produce, Dairy, Bakery, Meat)
- **View**: Active promotions in Promotions tab

### AI Product Comparison
- **Access**: From product details or search results
- **Compare**: Up to 3 products side-by-side
- **AI Recommendation**: See match scores and best value

### Checkout
- **From Home**: Tap "Checkout" (green button)
- **Payment Options**:
  - ✨ Automatic Checkout (Recommended)
  - 💳 Credit/Debit Card
  - 📱 Mobile Wallet
- **Features**: Real-time totals, tax calculation, savings display

## 📱 Screen Flow Examples

### Quick Shopping Flow
1. Home → Scan Item → Scan "123456789"
2. View Product → Add to Cart
3. Home → Checkout → Select Payment
4. Complete Purchase ✅

### Browse & Compare Flow
1. Home → Search → Type "Milk"
2. Tap product → View Details
3. Tap Compare → Select 3 products
4. Review AI Recommendation
5. Add to Cart → Checkout

### List Shopping Flow
1. Home → Start Shopping
2. Tap Scan Product
3. Scan multiple items
4. Return to List
5. Tap Checkout
6. Complete Purchase

## 💡 Pro Tips

1. **Cart Badge**: Watch the red badge on Cart icon for item count
2. **Cart Summary**: Home screen shows total when cart has items
3. **Quick Navigation**: Use back arrows or navigation bar
4. **Toast Notifications**: Green popups confirm cart additions
5. **Feature Guide**: Profile → Feature Guide for detailed help

## 🎨 Visual Indicators

- 🟢 **Green**: Primary actions (Add to Cart, Checkout, Confirm)
- 🔵 **Blue**: Search and navigation features
- 🟣 **Purple**: Scanning features
- 🟠 **Orange/Yellow**: Deals, promotions, secondary actions
- 🔴 **Red**: Cart badge, sale tags, remove actions
